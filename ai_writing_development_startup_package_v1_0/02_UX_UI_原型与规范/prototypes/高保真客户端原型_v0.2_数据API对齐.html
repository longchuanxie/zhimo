创建项目

表单字段与 Project 对象保持一致，project.type 决定后续导航、卡片类型和 Agent 身份。

动作映射POST /projects
Service: ProjectService.createProject(payload)
Table: projects + outlines
Side effect: create default outline for project
Default: workspace_id=default_workspace, created_by=default_user
project.type = research

研究 / 论文

默认卡片：excerpt、argument、term、citation。默认 Agent：research_assistant。

project.type = fiction

小说 / 长文

默认卡片：character、setting、event、plot。默认 Agent：fiction_editor。

project.type = free_writing

自由写作

默认卡片：inspiration、theme、image、fragment。默认 Agent：writing_editor。
