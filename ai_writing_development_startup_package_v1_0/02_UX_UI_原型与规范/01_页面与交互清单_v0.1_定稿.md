---

## 2. 客户端主布局

```text
左侧：项目导航
中间：主工作区
右侧：智能助手面板
```

---

## 3. 核心页面关系

```text
项目列表 → 创建项目 → 项目首页
资料库 → 资料详情 → 选中资料片段 → 保存为卡片 → 卡片库
卡片库 → 关联大纲节点 → 大纲页
大纲页 → 节点转文档 → 文档编辑页
文档编辑页 → 助手协作 → 保存为知识 → 知识库
文档 / 大纲 → 导出页 → 导出文件
```
