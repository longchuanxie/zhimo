# 项目开发基线

## 1. 产品定位

AI 原生写作项目系统是一个面向严肃写作者的本地优先写作客户端。

它不是：

```text
普通富文本编辑器
简单 AI 聊天框
云端协作文档
```

它是：

```text
AI 原生的长文本创作项目系统
```

核心抽象：

```text
一个项目 = 一个写作宇宙
```

一个项目包含：

```text
资料库
卡片
大纲
文档
知识库
助手对话
导出记录
模型配置
```

---

## 2. MVP 边界

MVP 必须完成：

```text
创建项目
导入资料
资料解析
资料摘录为卡片
创建大纲
大纲节点转文档
正文写作
右侧智能助手多轮协作
本次参考内容预览
助手建议可解释
AI 输出保存为卡片 / 知识
Markdown / Word 导出
模型服务商配置
```

MVP 不做：

```text
登录注册
账号体系
团队协作
权限系统
云同步
企业 SSO
OCR
本地模型
向量检索
复杂版本对比
插件市场
```

---

## 3. UI 基线

```text
中文 UI 优先
Heroicons 24px outline
桌面客户端三栏布局
左侧项目导航
中间主工作区
右侧智能助手
```

---

## 4. 技术基线

```text
客户端：Tauri 优先，Electron 可备选
前端：React + TypeScript
图标：Heroicons
本地数据库：SQLite
本地文件：项目库目录
模型接入：OpenAI-compatible
认证：后置
默认用户：default_user
默认工作空间：default_workspace
```
