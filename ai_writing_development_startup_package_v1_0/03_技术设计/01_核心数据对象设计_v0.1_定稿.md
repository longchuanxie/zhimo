## 2. 核心关系

```text
User
  └─ Workspace
      └─ Project
          ├─ Document
          ├─ Source
          │   └─ SourceChunk
          ├─ Card
          ├─ Outline
          │   └─ OutlineNode
          ├─ Knowledge
          ├─ AgentThread
          │   └─ AgentMessage
          ├─ AgentRun
          ├─ ContextPack
          └─ ExportTask
```

---

## 3. MVP 默认身份

```text
User.id = default_user
Workspace.id = default_workspace
```
