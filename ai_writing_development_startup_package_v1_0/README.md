# AI 原生写作项目系统定稿交付包

## 1. 包说明

本压缩包整理了本次会话中对后续开发有直接作用、且已经形成定稿方向的文档与 UI 原型。

产品形态已经明确为：

```text
桌面客户端 / 本地优先 / 单用户 MVP / 中文 UI 优先 / Heroicons 图标规范 / SQLite 本地存储
```

MVP 不包含登录、注册、账号体系和复杂权限，默认使用：

```text
default_user
default_workspace
```

## 3. 核心开发原则

```text
用户看到中文
研发看到映射
数据结构保持英文
AI 调用必须可追溯
本地数据优先保存
AI 输出不得直接覆盖正文
API Key 不得明文存储
```

---

## 4. 强制开发流程

所有开发任务必须遵循：

```text
需求领取 → 需求理解 → 方案设计 → 编码 → 编码审核 → 测试 → 进度更新
```

允许扩展为：

```text
需求领取 → 需求理解 → 风险识别 → 方案设计 → 方案评审 → 编码 → 自检 → 编码审核 → 测试 → 缺陷修复 → 验收 → 进度更新 → 文档归档
```
