## 3. 推荐开发顺序

```text
客户端底座
  ↓
SQLite / 文件系统
  ↓
项目 / 文档
  ↓
资料 / 卡片
  ↓
大纲 / 知识库
  ↓
模型 / Agent / ContextPack
  ↓
导出 / 任务中心 / 稳定性
```
