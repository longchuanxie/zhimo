# 编码审核清单

## 1. 功能审核

- [ ] 功能符合任务卡要求
- [ ] 未遗漏空状态
- [ ] 未遗漏错误状态
- [ ] 未遗漏加载状态
- [ ] 操作路径符合 UI 原型
- [ ] 不引入非 MVP 范围功能

---

## 2. 架构审核

- [ ] 组件不直接访问数据库
- [ ] 组件不直接访问文件系统
- [ ] 组件不直接调用模型 API
- [ ] 数据访问经过 Service
- [ ] AI 调用经过 AgentService / ContextService / ModelService
- [ ] 文件操作经过 FileService
- [ ] API Key 经过 SecretService

---

## 3. UI 审核

- [ ] 用户可见文案为中文
- [ ] 状态标签为中文
- [ ] 表单字段为中文
- [ ] 错误提示为中文
- [ ] 图标使用 Heroicons
- [ ] 无文字占位图标
- [ ] 无英文对象名暴露给普通用户

---

## 4. Agent 审核

- [ ] AI 调用前生成 ContextPack
- [ ] ContextPack 可预览
- [ ] 禁用资料不进入上下文
- [ ] 已废弃知识不进入上下文
- [ ] AgentRun 被记录
- [ ] AgentMessage 被保存
- [ ] AI 输出不直接覆盖正文
- [ ] 不展示原始思维链

---

## 5. 安全审核

- [ ] API Key 不明文保存
- [ ] API Key 不进入日志
- [ ] API Key 不进入错误提示
- [ ] 本地文件路径不暴露敏感信息
- [ ] 删除操作有确认
- [ ] 默认软删除

---

## 6. 测试审核

- [ ] 有正常路径测试
- [ ] 有空状态测试
- [ ] 有错误状态测试
- [ ] 有数据持久化测试
- [ ] 有重启恢复测试
- [ ] 有模型未配置测试
- [ ] 有本地文件失败测试
