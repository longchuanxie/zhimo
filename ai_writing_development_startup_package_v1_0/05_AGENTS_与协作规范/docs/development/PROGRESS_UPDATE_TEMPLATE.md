## 状态变更规则

```text
待开发 → 开发中
开发中 → 待联调
开发中 → 阻塞
待联调 → 待验收
待验收 → 已完成
阻塞 → 开发中
任意状态 → 后置
```

禁止：

```text
待开发 → 已完成
开发中 → 已完成
未测试 → 已完成
未审核 → 已完成
```
