# AI 原生写作项目系统功能优先级表 v0.1

## P0：MVP 必须实现

| 模块 | 功能 |
|---|---|
| 客户端底座 | 桌面客户端、本地项目库、三栏布局 |
| 项目系统 | 项目列表、创建项目、项目首页、项目设置 |
| 文档编辑 | 文档列表、新建文档、编辑器、自动保存、字数统计 |
| 资料库 | 导入文件、粘贴文本、资料解析、资料详情 |
| 卡片系统 | 新建卡片、资料摘录为卡片、卡片状态 |
| 大纲系统 | 大纲树、节点创建、节点排序、节点转文档 |
| 知识库 | 新建知识、编辑知识、从助手保存为知识 |
| 智能助手 | 多轮对话、绑定当前对象、快捷动作、助手输出操作 |
| 上下文 | 本次参考内容预览、ContextPack 快照 |
| 模型设置 | OpenAI-compatible 服务商配置、连接测试 |
| 导出 | Markdown、Word、导出记录 |
| 反馈 | Toast、任务中心、错误状态 |
| UI 规范 | 中文优先、Heroicons、空状态 |
