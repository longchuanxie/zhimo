# 导出实现方案 v1.0

## 1. MVP 导出格式

```text
Markdown
Word docx
```

---

## 2. 导出范围

```text
整个项目
当前文档
按大纲范围
```

研发枚举：

```text
whole_project
current_document
outline_scope
```

---

## 3. 导出流程

```text
选择导出范围
  ↓
选择导出格式
  ↓
预览导出顺序
  ↓
创建 ExportTask
  ↓
TaskService 执行导出
  ↓
生成本地文件
  ↓
更新 ExportTask.status
```

---

## 4. Markdown 导出

输入：

```text
Document.content_json
Document.plain_text
OutlineNode order
```

输出：

```text
.md
```

规则：

```text
heading → #
paragraph → 普通段落
blockquote → >
bulletList → -
orderedList → 1.
```

---

## 5. Word 导出

建议使用：

```text
docx
```

映射：

```text
heading → HeadingLevel
paragraph → Paragraph
blockquote → Paragraph with style
bulletList → bullet
orderedList → numbered
```

---

## 6. 文件路径

```text
projects/{project_id}/exports/{timestamp}_{title}.md
projects/{project_id}/exports/{timestamp}_{title}.docx
```

---

## 7. 导出任务状态

```text
pending
running
succeeded
failed
cancelled
```

---

## 8. 错误处理

| 场景 | UI |
|---|---|
| 没有可导出文档 | 还没有可导出的内容 |
| 文件写入失败 | 导出文件写入失败 |
| 格式不支持 | 暂不支持该导出格式 |
| 文档内容异常 | 文档内容包含暂不支持的结构 |

---

## 9. 可维护性要求

导出器按格式拆分：

```text
MarkdownExporter
DocxExporter
```

统一接口：

```ts
interface Exporter {
  export(input: ExportInput): Promise<ExportResult>
}
```
